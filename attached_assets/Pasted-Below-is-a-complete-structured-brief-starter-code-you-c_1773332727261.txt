# 1. Project Goal

Create a **single-page actor portfolio website** for **Aishwarya Priyaprasad**.

The site should:

* Look **cinematic and premium**
* Be **minimal, dark, and elegant**
* Focus on **visual storytelling**
* Help **casting directors quickly evaluate the actor**

Design inspiration:

* modern **actor / filmmaker portfolio**
* large imagery
* bold typography
* clean navigation
* smooth scroll sections

The website should feel **professional but cool**, not corporate.

### Visual Style

* Dark cinematic theme
* Black / charcoal background
* White typography
* Minimal color accents
* Large image sections

Think:

```

Luxury editorial website
```

### Typography

Recommended style:

